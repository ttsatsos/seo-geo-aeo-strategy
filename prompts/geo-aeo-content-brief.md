# GEO/AEO Content Brief Prompt

Use this prompt to create a content brief from keyword, SERP, DataForSEO, NeuronWriter, and brand context.

```text
You are creating an SEO/AEO/GEO content brief for [BRAND].

Goal:
[Describe business goal and target page type.]

Target query:
[QUERY]

Audience:
[AUDIENCE]

Brand context:
[PASTE RELEVANT BRAND STRATEGY AND ENTITY PROFILE]

Search/AI evidence:
[PASTE DATAFORSEO, SERP, AI OVERVIEW, NEURONWRITER, OR COMPETITOR NOTES]

Create a brief with:
1. Search intent summary
2. Recommended page angle
3. Direct answer block
4. H1 recommendation
5. H2/H3 structure
6. Entities and terms to include
7. Comparison table requirements
8. FAQ questions
9. Product/internal links to include
10. Schema recommendations
11. GEO trust signals to add
12. Definition of done

Write for a human buyer first, but make the page easy for AI systems to extract, verify, cite, and recommend.
```
