# Comparison Page Generator Prompt

```text
Create a comparison page draft for [BRAND].

Page title:
[TITLE]

Comparison type:
[X vs Y / best X for use case / price-tier / feature matrix / when to choose A over B / review roundup]

Audience:
[AUDIENCE]

Products or categories:
[LIST]

Brand facts and constraints:
[PASTE FACTS]

Evidence:
[PASTE REVIEWS, SPECS, DATAFORSEO, SERP, COMPETITOR, OR PRODUCT DATA]

Required structure:
1. Direct answer summary
2. Best choice table
3. Side-by-side comparison matrix
4. Use-case recommendations
5. Pros and cons
6. Price or cost guidance
7. Evidence/review themes
8. FAQs
9. Internal links
10. Schema notes

Make all claims specific, verifiable, and useful. Avoid unsupported superiority claims.
```
