# Linear Workflow

Linear is the execution layer. GitHub is the source of truth.

## Status Flow

1. Backlog
2. Ready for Human
3. Ready for Agent
4. Agent Working
5. Needs Human Review
6. Approved
7. Done

## Agent Rule

Agents should not mark strategy/content tasks done without human review. They should move tasks to `Needs Human Review` and leave a comment with:

- what changed
- where the artifact lives
- evidence used
- open assumptions
- recommended next step
