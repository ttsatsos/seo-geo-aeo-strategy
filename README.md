# Brand Search Intelligence

This repo is the canonical source of truth for SEO, AEO, and GEO strategy across managed brands.

It is designed to be readable by humans and agents. Keep durable strategy, prompts, templates, research notes, query sets, scoring rubrics, and brand context here. Use Linear for tasks and execution.

## Operating Principle

- **GitHub:** durable strategy, templates, prompts, datasets, decisions, reusable context.
- **Linear:** tasks, assignments, statuses, review loops, agent delegation.
- **DataForSEO:** keyword, SERP, AI Overview, competitor, and visibility data.
- **NeuronWriter:** article-level recommendations and content scoring while it remains useful.
- **LLM agents:** research, draft, audit, score, rewrite, and prepare changes for human review.

## Repo Map

- `brands/` - brand-specific strategy, entity profiles, content plans, and comparison page maps.
- `frameworks/` - reusable SEO/AEO/GEO models, scoring rubrics, and audit systems.
- `prompts/` - prompts for briefs, rewrites, audits, and comparison page generation.
- `dataforseo/` - query sets, tracking specs, API workflow notes, and reporting definitions.
- `linear/` - issue templates, workflow conventions, labels, and agent delegation rules.
- `templates/` - page templates and document templates.
- `operations/` - recurring workflows, QA checklists, and publishing process.
- `references/` - saved outside ideas, notes, and frameworks worth preserving.

## Core Strategy

Modern search visibility has three overlapping layers:

| Layer | Goal | Optimizes For | Measures |
|---|---|---|---|
| SEO | Discoverability | Crawlability, indexation, rank, metadata, internal links, content depth | Rankings, impressions, clicks, CTR, indexed pages |
| AEO | Extractability | Clean answers, definitions, FAQs, comparison tables, schema, direct response blocks | Featured snippets, PAA presence, answer extraction quality, AI Overview inclusion |
| GEO | Recommendability | Entity trust, citations, third-party validation, product data, original research, reviews | AI citations, AI share of voice, competitor citation share, recommendation frequency |

The goal is not just to rank. The goal is to become easy for search engines and AI systems to find, understand, trust, extract from, cite, and recommend.

## Working Rule

Every strategic page should answer four questions:

1. Can search systems find it?
2. Can answer systems extract from it?
3. Can generative systems verify it?
4. Can a buyer use it to make a decision?
