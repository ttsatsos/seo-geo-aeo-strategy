# Weekly SEO/AEO/GEO Review

## Inputs

- DataForSEO query tracking
- GSC performance
- NeuronWriter scores for active drafts
- AI visibility audit results
- Linear open issues
- published content changes

## Review Questions

1. Which pages gained or lost visibility?
2. Which prompts cite competitors instead of us?
3. Which pages need better extractability?
4. Which PDPs need fresher or more complete product data?
5. Which comparison pages should be created next?
6. Which off-site trust signals are missing?
7. Which tasks should be agent-ready?

## Outputs

- update relevant brand docs
- create or update Linear issues
- refresh query priorities
- record wins/losses in tracking files
